# Deploy Permanen (Render)

Project ini sudah disiapkan pakai `render.yaml`.

## Langkah cepat

1. Push project ini ke GitHub.
2. Buka Render: https://dashboard.render.com
3. Pilih **New +** â†’ **Blueprint**.
4. Hubungkan repo GitHub kamu.
5. Render akan membaca `render.yaml` otomatis.
6. Klik **Apply** untuk deploy.

## Hasil

- Kamu dapat URL tetap dari Render (contoh: `https://max-exc-chat.onrender.com`).
- App tetap online tanpa harus menyalakan laptop.

## Catatan penting

- Paket free biasanya bisa sleep saat tidak ada traffic beberapa menit.
- Untuk benar-benar selalu aktif 24/7 tanpa sleep, upgrade plan di Render atau pakai VPS.
